About This Dashboard

This dashboard shows anonymous visitor statistics based only on geolocation (country, region, city). No personal or identifying information is collected—only approximate location from IP address.

Visits are recorded by a Netlify Function (IP geolocation), stored in Supabase, and aggregated weekly by a GitHub Action. The dashboard updates every Monday after the sync runs.
